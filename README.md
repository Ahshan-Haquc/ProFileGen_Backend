# ProFileGen (SaaS Platform) Backend

## Overview

ProFileGen is a backend service built to support a CV / profile generation platform. It is a SaaS application. It provides authentication, user management, admin controls, subscription handling, and content uploading so end users can create and manage their professional profiles securely.
<br><br>
<br>
Live link of the project: https://profilegen-frontend.vercel.app/  <br>
Github repository link of the frontend: https://github.com/Ahshan-Haquc/ProFileGen_Frontend


## Key features

- User authentication and authorization
- Admin dashboard support for managing application content
- Subscription and payment integration
- CV generation limit checks and usage controls
- Secure file upload handling
- Email notifications for user workflows
- Structured TypeScript codebase for maintainability.

## Tech stack

- Node.js
- TypeScript
- Express
- MongoDB database
- Cloudinary for file uploads
- Stripe for subscription processing

## Get started
1. clone the repository
2. Install dependencies: `npm install`
3. Configure environment variables
4. Start the server: `npm nun dev`

## Notes

This repo is the backend application for the ProFileGen platform. This backend is connected with a separate React + TypeScript + Tailwind CSS + Shadcn/UI frontend. https://github.com/Ahshan-Haquc/ProFileGen_Frontend
